jquery.spriteAnimation examples

Multiple Sequences in a single sheet

This animation uses the sprite sheet below. It contains both animations on different rows. 

$('#anim')
	.spriteAnimation({
		addSequences:{
			a: {
				duration: 21
			},
			b: {
				duration: 21,
				firstFramePosition: {x:0, y:25}
			}
		},
		setSequence: "a"
	}
)
